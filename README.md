# support_frontend_for_dearhero

## SCSS自動反映コマンド

```unix
npm run all
```

## npm install コマンド

```unix
npm install -y
```


## ディレクトリ構成

```
- src/assets
  - css: コンパイル後のcssが作成される
  - html: コンパイル後のhtmlが作成される
  - scss
    - base: サイト共通定義ディレクトリ
    - mixin: サイト内で何度も使うデザイン定義ディレクトリ
    - module: パーツを定義するディレクトリ
    - page: ページ専用CSS 
    - setting: 変数定義ディレクトリ 

```
